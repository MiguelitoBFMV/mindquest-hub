# 🧠 MindQuest Hub

**Turn chaos into progress.**

MindQuest Hub es una plataforma personal orientada a organizar y visualizar el progreso en hobbies como videojuegos, anime, manga y música, con un enfoque en estadísticas, seguimiento y toma de decisiones.

---

## 🚀 Estado del proyecto

🟡 En desarrollo (fase inicial)

Actualmente el proyecto se encuentra en etapa de construcción del **frontend base (HTML + CSS)**, con datos estáticos (hardcoded) como parte del proceso de aprendizaje y prototipado.

---

## 🎯 Objetivo

Crear una plataforma centralizada que permita:

- 📊 Visualizar progreso en distintos hobbies
- 🎮 Gestionar backlog de videojuegos
- 📺 Seguir anime y manga
- 🎵 Integrar estadísticas musicales (Last.fm)
- 🧠 Generar sugerencias inteligentes (Brain Suggestion)
- 📈 Analizar hábitos de consumo

---

## 🧱 Estructura actual

El proyecto actualmente contiene:

- Layout principal tipo dashboard
- Sidebar de navegación
- Topbar con búsqueda y perfil
- Secciones principales:
  - Brain Suggestion
  - Currently Active
  - Hub Analytics
  - Recent Activity
  - Audio Hub

## 🧪 Metodología de desarrollo

El desarrollo se está realizando de forma progresiva:

1. Maquetación HTML (estructura)
2. Estilos CSS (layout + diseño)
3. Interactividad con JavaScript
4. Migración a React
5. Integración con backend
6. Persistencia en base de datos

---

## 🧠 Conceptos clave del proyecto

- Separación entre UI y datos
- Componentización (pensado para React)
- Diseño modular (escalable a múltiples secciones)
- Enfoque en visualización y estadísticas

---

## 📌 Notas

- Los datos actuales son simulados (hardcoded)
- La estructura está pensada para evolucionar hacia una SPA
- El diseño está inspirado en dashboards modernos con estilo dark UI
